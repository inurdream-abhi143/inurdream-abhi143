# ChainStrap — Cross-Platform Web & Mobile App

Overview

ChainStrap demonstrates a shared-business-logic approach where web (Next.js) and mobile (React Native) applications reuse core services and validation logic for consistency.

Features

- Shared modules for auth, API clients, and input validation
- Consistent UI components across web and mobile
- Secure authentication and token sync

Tech

Next.js, React Native, Node.js, MongoDB

Local setup (dev)

1. Clone the repo
2. Install web and mobile dependencies
3. Run dev servers for each platform
