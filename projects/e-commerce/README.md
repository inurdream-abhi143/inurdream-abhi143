# E-Commerce Website with Admin Panel

Overview

A responsive e-commerce frontend with an admin panel for managing products and orders. Built to demonstrate typical online store flows: product browsing, cart & checkout, and backend admin features.

Features

- Product listing with filters and search
- Shopping cart, order creation and mock checkout (JSON Server)
- Admin panel with role-based access control (RBAC)
- Protected routes and client-side validation

Tech

React, Context API, Bootstrap, JSON Server, HTML/CSS, JavaScript (ES6+)

Local setup (dev)

1. Clone the repo
2. cd e-commerce
3. npm install
4. npm run dev
