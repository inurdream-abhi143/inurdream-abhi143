# Blogify — Full Stack Blog Platform

Overview

Blogify is a full-stack blogging platform with role-based authentication, publishing workflow, comments, and secure REST APIs. Built as a learning project to demonstrate end-to-end features.

Features

- User registration & JWT-based authentication
- Role-based publishing (author, editor, admin)
- Commenting system and moderation
- RESTful APIs with input validation

Tech

React, Node.js, Express, MongoDB, JWT, Mongoose

Local setup (dev)

1. Clone the repo
2. cd blogify
3. npm install
4. Start backend and frontend (see project README for scripts)
